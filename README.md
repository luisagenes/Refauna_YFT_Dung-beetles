# yft_ag_dung-beetle_tnp.xlsx 
Comparison between dung beetle communities attracted to feces of yellow-footed tortoises (C. denticulatus) and howler monkeys (A. guariba) in Tijuca National Park (Rio de Janeiro, Brazil. Data for dung beetle attraction to howler monkey feces is available in  Genes et al. (2019). Genes et al. (2019) had a higher sampling effort (current study = 42; Genes et al. 2019 = 78). 
   
    metadata
    fecal_source: vertebrate species feces used as bait to capture dung beetles
    Dung_beetle_species: species of dung beetle collected
    n_individuals: number of individuals collected
    relative_abundance: relative abundance (%) of dung beetle species in samples collected using the same fecal source

# yft_dung-beetle_tnp.xlsx
Ecological characteristics of dung beetle species attracted by yellow-footed tortoise feces in Tijuca National Park, Rio de Janeiro, Brazil. Dung beetles were collected across twenty-one traps placed along two transects during two sampling periods.
```
  metadata
  Tribe: dung beetle species tribe
  Species: dung beetle species
  n_individuals: number of individuals collected
  Activity: N = nocturnal, D = diurnal, U = unknown
  Behavior: T = telecoprid, P = paracoprid, E = endocoprid, unk = unknown
  Feeding guild: C = coprophage, N = necrophage, G = generalist, U = unknown
```
